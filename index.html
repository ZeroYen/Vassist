Introduction

VAssist is a lightweight, multilingual virtual assistant designed to enhance user interaction on websites. Built with modern web technologies, it provides a natural conversation interface with support for multiple languages and customizable themes.

Key Features:
  • Advanced dialogue system with deep nesting
  • Support for 6+ languages with rich message styling
  • Multiple theme options (Light/Dark/Custom)
  • AI capabilities and automation tools
  • Context-aware responses
  • No external dependencies

Installation

NPM

Install VAssist using npm:

npm install vassist --save

CDN

Include VAssist directly from a CDN:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/vassist/dist/styles.css"> <script src="https://cdn.jsdelivr.net/npm/vassist/dist/script.js"></script>

Manual Download

Download the latest version from GitHub:

  1. Download the latest release
  2. Include the files in your project
  3. Add the necessary HTML tags
Note: Make sure to include both the CSS and JavaScript files in your HTML.

Quick Start

Get started with VAssist in three simple steps:

1. Include Required Files

<!-- Add to your HTML head -->
<link rel="stylesheet" href="path/to/vassist/styles.css">

<!-- Add before closing body tag -->
<script src="path/to/vassist/script.js"></script>

2. Initialize VAssist

// Basic initialization
const assistant = new VirtualAssistant();

// With custom options
const assistant = new VirtualAssistant({
    language: 'en',
    theme: {
        primaryColor: '#2196f3',
        textColor: '#333333'
    }
});

3. Add Language Data

Create a JSON file for each supported language:

{
  "welcome": {
    "message": {
      "text": "Welcome! How can I help?",
      "style": "greeting"
    },
    "options": [
      {
        "text": "Get Started",
        "next": "start",
        "style": "primary"
      },
      {
        "text": "Learn More",
        "next": "about",
        "style": "secondary"
      }
    ]
  }
}

Adding a New Language

To add a new language:

  1. Create a new JSON file in the languages directory
  2. Follow the language template structure
  3. Register the language in your configuration
// Register a new language
assistant.addLanguage('fr', {
    welcome: {
        message: "Bonjour! Comment puis-je vous aider?",
        options: [
            {text: "Commencer", next: "start"},
            {text: "En savoir plus", next: "about"}
        ]
    }
});

Theming

VAssist supports comprehensive theming through CSS variables and the template.json configuration.

CSS Variables

:root {
    --vassist-primary-color: #2196f3;
    --vassist-text-color: #333333;
    --vassist-bg-color: #ffffff;
    --vassist-button-color: #2196f3;
    --vassist-button-text-color: #ffffff;
}

Custom Themes

Create custom themes by overriding CSS variables:

/* Dark theme */
.vassist-dark-theme {
    --vassist-bg-color: #1a1a1a;
    --vassist-text-color: #ffffff;
    --vassist-primary-color: #64b5f6;
}
Tip: Use CSS custom properties for dynamic theme switching without JavaScript.

Dialogue Structure

The dialogue system supports deep nesting with context-aware responses:

{
  "dialogues": {
    "start": {
      "message": {
        "text": "How can I assist you today?",
        "style": "question"
      },
      "options": [
        {
          "id": "customize",
          "text": "Customize VAssist",
          "response": {
            "text": "Let's personalize VAssist to your preferences",
            "style": "helpful"
          },
          "nextNode": "customize_options"
        }
      ]
    },
    "customize_options": {
      "message": {
        "text": "What would you like to customize?",
        "style": "helpful"
      },
      "options": [
        {
          "id": "theme",
          "text": "Change Theme",
          "nextNode": "theme_selection"
        }
      ]
    }
  }
}

API Reference

Core Methods

Method Parameters Return Description
open() None void Opens the chat window
close() None void Closes the chat window
setLanguage(lang) string void Changes the current language
addLanguage(code, data) string, object void Adds a new language

Event Handling

// Listen for events
assistant.on('message', (message) => {
    console.log('New message:', message);
});

assistant.on('languageChange', (lang) => {
    console.log('Language changed to:', lang);
});

assistant.on('open', () => {
    console.log('Assistant opened');
});

Events

VAssist provides a rich event system for tracking and responding to user interactions.

Available Events

Event Data Description
open none Fired when the chat window opens
close none Fired when the chat window closes
message Message object Fired when a message is sent/received
languageChange Language code Fired when language is changed

Custom Event Handling

// Custom event handler
assistant.on('message', (message) => {
    // Track analytics
    analytics.track('chat_message', {
        type: message.type,
        content: message.text
    });

    // Custom logging
    console.log(`[${message.timestamp}] ${message.text}`);
});
Best Practice: Always remove event listeners when they're no longer needed to prevent memory leaks.
